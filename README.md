## Tested on:
- Chrome 51

## Tasks
- `npm run build` - production build
- `npm run build --debug` - development build
- `npm run build --watch` - watch files and rebuild on change
- `npm run lint` - run ESLint
- `npm test` - run JS unit tests
- `npm test --watch`
- `npm test --coverage`
